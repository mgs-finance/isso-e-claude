Aula 09.1 Visão Geral ⏱ ~20 min

A Anthropic Academy
— O que é e como usar

A Anthropic criou uma plataforma gratuita de cursos com certificado oficial. São 15 cursos em inglês. Este módulo é o guia completo em português — o que cada curso ensina, para quem serve e em qual ordem estudar.

O que é a Anthropic Academy

Em 2025, a Anthropic lançou sua própria plataforma de aprendizado hospedada em anthropic.skilljar.com. São cursos autoguiados, completamente gratuitos, com certificado oficial da Anthropic ao final. Não é necessário ter plano pago do Claude para acessar — apenas criar uma conta.

Os cursos cobrem desde o uso básico do Claude até construção de servidores MCP em Python e desenvolvimento com o Claude Code. O conteúdo é denso, técnico em vários pontos, e representa o pensamento oficial da equipe que construiu o Claude.

📍

Como acessar

Acesse anthropic.skilljar.com e crie uma conta gratuita. Todos os cursos listados neste módulo estão disponíveis sem custo. O certificado é emitido automaticamente ao concluir cada curso.

Por que ler a fonte oficial mesmo sendo em inglês

Os cursos gratuitos da Anthropic trazem o pensamento original da equipe que construiu o Claude: como eles querem que a ferramenta seja usada, quais são os limites reais, quais as práticas validadas internamente. Nenhum curso de terceiros substitui isso — só complementa.

💡

Estratégia ideal de estudo

Use o Isso é Claude como sua base principal — em português, com contexto brasileiro, com casos reais de negócio. Use a Anthropic Academy como aprofundamento técnico, especialmente nos cursos de MCP avançado e Claude Code que não têm equivalente completo em nenhum curso pago do mercado.

Curso 2 — AI Fluency: Framework & Foundations

🧠
✓ Gratuito + Certificado Iniciante
AI Fluency: Framework & Foundations
anthropic.skilljar.com/ai-fluency-framework-foundations
Acessar →

Desenvolvido em parceria com o Ringling College, este curso não é sobre nenhuma ferramenta específica — é sobre como pensar estrategicamente sobre inteligência artificial. O objetivo é construir uma base conceitual sólida: entender o que a IA pode e não pode fazer, como colaborar com ela de forma ética e eficaz, e como avaliar criticamente os outputs que ela produz.

O que você aprende
Como funcionam os modelos de linguagem em termos conceituais — sem precisar saber programar
O framework "4E" da Anthropic: como usar IA de forma Eficaz, Eficiente, Ética e Segura
Como identificar quando a IA está errando ou sendo imprecisa — o conceito de alucinação e como mitigá-lo
Como colaborar com IA de forma que preserve o pensamento crítico humano, sem criar dependência
Questões de privacidade, viés e responsabilidade no uso de IA no ambiente profissional
Como apresentar e justificar o uso de IA para colegas céticos ou conservadores
✅ Relação com o Isso é Claude
Complementa o Módulo 01 (Como o Claude Pensa) com uma perspectiva acadêmica e ética mais aprofundada. Recomendado especialmente para quem vai usar IA em educação, saúde, direito ou serviços públicos.

Curso 3 — Introdução ao Claude Cowork

🖥️
✓ Gratuito + Certificado Iniciante ⭐ Exclusivo Academy
Introdução ao Claude Cowork
anthropic.skilljar.com/introduction-to-claude-cowork
Acessar →

O Cowork é a funcionalidade que transforma o Claude em um agente de desktop completo: ele acessa arquivos do seu computador, organiza pastas, cria documentos, gera apresentações e mantém memória entre sessões. Este curso cobre como começar a usar o Cowork de forma produtiva já na primeira semana, incluindo o sistema de tarefas agendadas e plugins.

O que você aprende
Como instalar e configurar o app do Claude Cowork no seu computador (macOS e Windows)
O task loop do Cowork: como o Claude planeja, executa e revisa tarefas em etapas
Como configurar tarefas recorrentes e agendamentos automáticos (ex.: relatório toda sexta às 17h)
Como usar plugins do Cowork para expandir o que ele pode fazer com seus arquivos
Como trabalhar com o Cowork em fluxos de pesquisa e síntese de documentos longos
Como dirigir o agente de forma responsável — quando interromper, ajustar e retomar tarefas
⭐ Conteúdo exclusivo — recomendado!
O Módulo 11 do Isso é Claude cobre o Cowork em profundidade, mas este curso traz a perspectiva oficial da equipe da Anthropic com detalhes de configuração que complementam o que você aprendeu. Vale fazer os dois.

Curso 4 — AI Automation with Claude (Coursera)

AI Automation with Claude
coursera.org/learn/ai-for-everyone-ai-automation-with-claude

Como usar RAG básico com documentos: subir bases de conhecimento e fazer o Claude responder com base nelas
Como aplicar automação além de texto — imagens, tabelas e outros formatos multimodais como input
Como usar IA para tarefas pessoais e de carreira, não só profissionais
Perspectiva interna da Anthropic sobre os casos de uso que realmente funcionam em produção
🔵 Aprofunda o Isso é Claude
Complementa o Módulo 04 (Projects & Memory) com a visão de quem estava dentro da Anthropic. O ângulo de Dan Mellott sobre "automação que escala" é diferente do que qualquer curso externo oferece.

Curso 7 — Introdução ao Agent Skills

🧩
✓ Gratuito + Certificado Intermediário
Introdução ao Agent Skills
anthropic.skilljar.com/introduction-to-agent-skills
Acessar →

Skills são o sistema de instruções reutilizáveis do Claude Code — você cria um arquivo SKILL.md e o Claude passa a carregar essas instruções automaticamente sempre que a tarefa combina com o contexto da Skill. Este curso ensina a criar, configurar, compartilhar e depurar Skills para uso individual e distribuição em equipes.

O que você aprende
A estrutura exata de um arquivo SKILL.md e como o Claude decide quando ativá-la
Como criar uma Skill do zero para uma tarefa específica do seu trabalho
Como combinar múltiplas Skills para fluxos de trabalho de múltiplas etapas
Como distribuir Skills para toda a equipe — versionar, compartilhar e atualizar centralmente
Como depurar quando uma Skill não está sendo ativada corretamente
Boas práticas de nomenclatura e organização de Skills em projetos grandes
🔵 Aprofunda o Isso é Claude
O Módulo 11 do Isso é Claude traz um gerador interativo de SKILL.md. Este curso da Anthropic ensina os detalhes de distribuição e debugging que vão além do básico.

Curso 9 — Introdução ao Model Context Protocol

🔌
✓ Gratuito + Certificado Intermediário ⭐ Recomendado
Introdução ao Model Context Protocol (MCP)
anthropic.skilljar.com/introduction-to-model-context-protocol
Acessar →

O MCP (Model Context Protocol) é o protocolo aberto criado pela Anthropic para conectar o Claude a qualquer serviço ou fonte de dados externa de forma padronizada. Este curso ensina a construir servidores e clientes MCP em Python do zero, dominando as três primitivas fundamentais do protocolo: ferramentas, recursos e prompts.

O que você aprende
O que é o MCP e por que ele existe — o problema que resolve na integração de IA com sistemas externos
As três primitivas do MCP: tools (ações que o Claude pode executar), resources (dados que o Claude pode ler) e prompts (templates reutilizáveis)
Como construir um servidor MCP em Python do zero que expõe ferramentas ao Claude
Como construir um cliente MCP que se conecta ao seu servidor
Como testar e depurar servidores MCP localmente antes de conectar ao Claude Code
Casos de uso reais: conectar o Claude a bancos de dados, APIs internas e sistemas legados via MCP
⭐ Aprofunda muito além do Isso é Claude
O Módulo 07 do Isso é Claude cobre como usar servidores MCP existentes. Este curso ensina como criar seus próprios servidores MCP — uma habilidade muito mais rara e valiosa.

Curso 10 — MCP: Tópicos Avançados

🚀
✓ Gratuito + Certificado Avançado ⭐ Para devs experientes
MCP: Advanced Topics
anthropic.skilljar.com/model-context-protocol-advanced-topics
Acessar →

A continuação direta do curso de introdução ao MCP, voltada para engenheiros que já dominam o básico e querem implementar servidores MCP em ambiente de produção. Cobre os padrões mais avançados que a maioria dos recursos online ainda não documenta bem — como notificações em tempo real, controle de acesso granular e mecanismos de transporte customizados.

O que você aprende
Sampling: como permitir que servidores MCP façam requisições de volta ao Claude para processamento adicional
Notificações em tempo real: como implementar um sistema de alertas entre o servidor MCP e o cliente
Controle de acesso ao sistema de arquivos: políticas granulares de permissão para acesso a arquivos via MCP
Mecanismos de transporte: HTTP, SSE (Server-Sent Events) e stdin/stdout — quando usar cada um
Padrões de segurança para servidores MCP em produção — autenticação, autorização e auditoria
Como escalar um servidor MCP para múltiplos usuários simultâneos
⭐ Sem equivalente no Isso é Claude
Este é o conteúdo mais técnico e avançado disponível gratuitamente sobre MCP. Se você quer construir infraestrutura de IA de nível profissional, este curso não tem substituto.
